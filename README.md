# Smart Grocery Store

A modern, full-stack e-commerce platform built with React, Node.js, and MongoDB.

## Features

- **Storefront**: Browse products by category, search, and view details.
- **Cart & Checkout**: Secure cart management and checkout with multiple payment options (COD, UPI).
- **Authentication**: Secure login using Email, Google, or Phone Number (OTP).
- **Admin Dashboard**: Real-time sales analytics, order management, and user/vendor management.
- **Vendor System**: Approval-based vendor registration and product management.

## Tech Stack

- **Frontend**: React.js, Framer Motion, Lucide Icons, Tailwind-inspired Vanilla CSS.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Authentication & Hosting**: Firebase.

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   cd backend && npm install
   ```
3. **Setup environment variables**:
   Set up your `.env` files in both root and backend directories with your Firebase and MongoDB credentials.
4. **Run the application**:
   ```bash
   # Start backend
   cd backend && npm start
   # Start frontend (from root)
   npm start
   ```
